projectMM

Plug in your ESP32. Pick the board. Flash. Open in a browser.

This browser can't flash directly. ESP Web Tools needs Web Serial — open this page in Chrome, Edge, or Opera on a desktop. Firefox and Safari don't support Web Serial. The release page on GitHub has the same binaries for manual flashing.

The Ethernet builds were verified on the Olimex ESP32-Gateway (LAN8720 PHY, reset on GPIO 5, MDIO addr 0). Boards with the same PHY but different pins (e.g. WT32-ETH01: reset on GPIO 16) need a local rebuild today.

After flashing — WiFi builds:
  1. The device boots a SoftAP named projectMM-xxxx.
  2. Join it from your phone or laptop.
  3. Open http://192.168.4.1 in a browser, enter your home WiFi credentials.
  4. The device reboots, joins your network, and the UI is reachable at its DHCP-assigned IP (visible in your router).
After flashing — Ethernet builds:
  1. Plug an Ethernet cable into the device.
  2. Open the DHCP-assigned IP in a browser (check your router for new devices). mDNS hostnames work too on most networks.

Source code and releases: github.com/ewowi/projectMM.
